用户正在开发一个小程序叫"游迹"，主要功能包括：
1. 记录游泳选材班学员的成绩数据
2. 记录学员的出勤记录
3. 记录学员的陆上动作打卡数据
4. 计划做"游迹朋友圈"功能，从教练侧收集更多数据
5. 赛事数据手动批量上传，用户可丰富学员成绩

【AI能力需求 - 2025年新增】
- 游迹中需要新增AI能力
- 教练侧收集的信息 → AI的养料
- 学员侧收集的信息 → AI的养料
- 两边数据汇聚后，作为AI模型训练或推理的数据基础

【新增功能规划 - 2026年6月】
以下四个方向已确认推进：

1. **训练打卡 + 成就徽章体系**
   - 学员主动打卡训练，连续打卡解锁徽章（如"早起练""全勤周""水下憋气王"等）
   - 类似Keep的激励机制，调动学员自驱力

2. **语音转文字 → 训练日志**
   - 教练每天训练完说30秒语音，AI转成结构化训练日志
   - 降低教练分享专业知识的门槛
   - 例：自动生成"今日主项：自由泳打腿，重点纠正了XX的呼吸节奏"

3. **专业问答榜 / 积分体系**
   - 家长在"游迹朋友圈"提问 → 教练回答 → 获得"专业积分"
   - 积分可兑换：赛事实名推荐名额、训练器材、平台曝光
   - 教练回答越多，平台专业可信度越高，形成正向循环

4. **学员进步报告（一键生成）**
   - AI自动汇总学员近期成绩、出勤、动作评分等数据
   - 生成排版精美的《月成长报告》，教练署名后一键转发家长
   - 教练分享了专业知识，家长看到了专业度，双方受益
